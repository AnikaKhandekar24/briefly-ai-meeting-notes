# Briefly

Briefly turns pasted meeting transcripts into structured summaries, decisions,
action items, topics, and follow-up emails.

## Features

- Local transcript analysis with no API key required
- Editable meeting summaries and action items
- Follow-up email drafting
- Persistent profile and workspace preferences
- Searchable saved-meeting library
- Markdown export, dark mode, and responsive design

## Run locally

Open `index.html` directly, or serve the directory with any static web server:

```bash
python -m http.server 8765
```
